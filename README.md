### How to Use:

1. Clone the repository:

   ```bash
   git clone https://github.com/kensdy/Twitter-Data-Extraction.git
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the main script:

   ```bash
   python main.py
   ```
