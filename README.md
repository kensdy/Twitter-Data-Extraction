### Como Usar:

1. Clone o repositório:

   ```bash
   git clone https://github.com/kensdy/Twitter-Data-Extraction.git
   ```

2. Instale as dependências:

   ```bash
   pip install -r requirements.txt
   ```

3. Execute o script principal:

   ```bash
   python main_pt.py
   ```
